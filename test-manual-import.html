Implementation Details

Algorithm: Hybrid Error Diffusion + Viterbi

How it works:

  • Resizes input image to 280×192
  • Processes each scanline with Floyd-Steinberg error diffusion
  • For each byte, tests 48 canonical patterns × 2 hi-bit settings (96 evaluations)
  • Calculates error using actual NTSC-rendered colors
  • Selects pattern with minimum perceptual color distance
  • Propagates quantization error to neighboring pixels

Performance:

  • Expected conversion time: 400-800ms for 280×192 image
  • Memory usage: ~10MB
  • Quality: NTSC-accurate with smooth gradients (no banding)
